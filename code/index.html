Stack overflow

Stack overflow is a website commonly used for discussion about issues within codeing. Users can submit questions or answer other peoples questions about topics within programming. There is also a function where the users of the website can vote vote for the best solutions to a specific problem. It's a good place to find solutions for the questions or issues within programming that you most definetly will have.

Fun fact about

The name of the website originate from a common error within software engineering which is caused by a program at tempting to use more stacks than is avaliable which results in a "stack overflow" error. It is rumoured that the particular name is chosen to indicate that the website is where programmers should go when their stack has overflowed, which in other words means when the programmer is out of ideas and need help.
